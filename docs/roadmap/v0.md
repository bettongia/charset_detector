# v0

## Charset detection utility [✓ Complete]

[Plan](../plans/completed/plan_initial_implementation.md)

Implement `betto_charset_detector` as a standalone, pure-Dart package that
detects the character encoding of an arbitrary byte sequence. The detector uses
a three-stage pipeline — BOM inspection, UTF-8 structural validation, and
candidate probing via the `charset` package — and supports all Dart and Flutter
target platforms (mobile, web, desktop) with no native dependencies.
